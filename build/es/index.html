NetworkX es un paquete de Python para la creación, manipulación y estudio de la estructura, dinámica y funciones de redes complejas.

Software para redes complejas


  • Estructuras de datos para grafos, dígrafos y multigrafos
  • Estructura de la red y medidas de análisis.
  • Estructura de la red y medidas de análisis
  • Generadores de grafos clásicos, grafos aleatorios y redes sintéticas
  • Los nodos pueden ser "cualquier tipo de objeto" (por ejemplo, texto, imágenes, registros XML)
  • Las aristas pueden contener datos arbitrarios (por ejemplo, pesos, series de tiempo)
  • Licencia de código abierto BSD de 3 cláusulas
  • Probado con más del 90% de cobertura de código
  • Los beneficios adicionales de Python incluyen la creación rápida de prototipos, la facilidad a la hora de enseñar a programar y la posibilidad de escribir código multiplataforma
