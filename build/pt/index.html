NetworkX é um pacote Python para a criação, manipulação e estudo da estrutura, dinâmica e funções de redes complexas.

Software para redes complexas


  • Estruturas de dados para gráficos, dígrafos e multigrafos
  • Muitos algoritmos padrão para grafos
  • Estrutura de redes e medidas de análise
  • Geradores para grafos clássicos, grafos aleatórios e redes sintéticas
  • Nós podem ser "qualquer coisa" (por exemplo, texto, imagens, registros XML)
  • Arestas podem conter dados arbitrários (por exemplo, pesos, séries temporais)
  • Open source com licença BSD 3 cláusulas
  • Bem testado com mais de 90% de cobertura de código
  • Benefícios adicionais do Python incluem prototipagem rápida, facilidade de ensinar e suporte a várias plataformas
