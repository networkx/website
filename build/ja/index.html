NetworkX は、複雑ネットワークの「構造・ダイナミクス・機能」の、「作成・操作・研究」のための Python パッケージです。

複雑ネットワークのためのソフトウェア


  • グラフ・有向グラフ・多重グラフのためのデータ構造
  • 多くの標準的なグラフアルゴリズム
  • ネットワーク構造と解析指標
  • 古典的グラフ、ランダムグラフ、合成ネットワークの生成器
  • ノードは "何にでも" なり得る (例: テキスト、画像、XML レコード)
  • エッジは任意のデータを保持可能（例：重み、時系列）
  • オープンソース 3-clause BSD ライセンス
  • 90%以上のコードカバレッジで十分にテスト済み
  • Python による追加の利点：高速なプロトタイピング、教育が容易、マルチプラットフォーム対応
