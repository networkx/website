Quick Example
-------------

>>> import networkx as nx
>>> G=nx.Graph()
>>> G.add_node("spam")
>>> G.add_edge(1,2)
>>> print(G.nodes())
[1, 2, 'spam']
>>> print(G.edges())
[(1, 2)]

See more `simple and complicated examples <http://networkx.github.io/documentation/latest/examples/>`_, or
a `gallery <http://networkx.github.io/documentation/latest/gallery.html>`_.
of network drawings.
